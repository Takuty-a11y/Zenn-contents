# はじめに

Zenn 記事を書き始めるにあたって、ローカル環境で Git プッシュすると自動で記事を投稿できる仕組みがあるということを知り環境構築しました。
Zenn 記事はmdで記載するため自分のテキストエディタでマークアップしてければ手軽に記事が書けます。
テキストエディタで書ければかなり便利なので環境構築手順を残すことにしました。

# 手順

- Zenn と GitHub リポジトリ連携
- Zenn CLI をインストール
- 記事作成とアップロード
